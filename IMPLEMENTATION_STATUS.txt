USAGE
Generating a Report:
1. Go to home screen
2. Tap "Farm Report" button in Quick Actions
3. Wait for "Generating Farm Report Card…" toast
4. Share dialog opens
5. Choose where to share/save

Customizing Farm Name:
1. Tap farm name in AppBar
2. Edit dialog opens
3. Type custom name (max 40 chars)
4. Tap Save
5. Name persists on device
